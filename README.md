# La Folloneta del Mago 🎩✨

Landing page para la actuación de magia en vivo en una furgoneta única.

## Características

- 🎪 Experiencia mágica íntima en un espacio único
- ✨ Ambiente profesional y acogedor
- 🚐 Furgoneta completamente equipada con las mejores prestaciones
- 🪄 Magia de calidad y cercana

## Tecnología

Este proyecto está construido con [Astro](https://astro.build), un framework web moderno y rápido.

## Comandos

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Construir para producción
npm run build

# Preview de la build
npm run preview
```

## Desarrollo

El servidor de desarrollo se inicia en `http://localhost:4321`
